1. 字符串对象处理

Thymeleaf 主要使用 org.Thymeleaf.expression.Strings 处理字符串

1. toString

toString 方法: [[${#strings.toString(user)}]]

2. length

length 方法: [[${#strings.length(user)}]]

2. 默认和空值处理

1. 非空处理 isEmpty

isEmpty 方法: [[${#strings.isEmpty(user)}]]

2. 默认值处理 defaultString

defaultString 方法: [[${#strings.defaultString(user.department, '无部门')}]]

3. 包含判断

1. 包含判断 contains

contains 方法: [[${#strings.contains(user.name, 'Luna')}]]

2. 包含判断==>忽略大小写 containsIgnoreCase

containsIgnoreCase 方法: [[${#strings.containsIgnoreCase(user.name, 'Luna')}]]

3. 前面包含 startsWith

startsWith 方法: [[${#strings.startsWith(user.name, 'L')}]]

4. 后面包含 endsWith

endsWith 方法: [[${#strings.endsWith(user.name, 'a')}]]

4. 截取替换

1. 单个查找 indexOf 没有则返回-1

indexOf 方法: [[${#strings.indexOf(user.name, 'x')}]]

2. 截取 substring 截取1~3

substring 方法: [[${#strings.substring(user.name, 1, 3)}]]

3. 截取 substringAfter 截取'L'之后的

substringAfter 方法: [[${#strings.substringAfter(user.name, 'L')}]]

4. 截取 substringBefore 截取'a'之前的

substringBefore 方法: [[${#strings.substringBefore(user.name, 'a')}]]

5. 替换 replace 将'L' ==> 'l'

replace 方法: [[${#strings.replace(user.name, 'L', 'l')}]]

5. 追加字符串

1. 追加 prepend 在Luna之前加一个op

prepend 方法: [[${#strings.prepend(user.name, 'op')}]]

2. 追加 append 在Luna之后加一个op

append 方法: [[${#strings.append(user.name, 'op')}]]

3. 连接 concat Lina op od 连接起来

concat 方法: [[${#strings.concat(user.name, 'op', 'od')}]]

4. 连接第一个值以后的参数 如果值为空 用第一个值代替 concatReplaceNulls Lina null od null 连接起来

concatReplaceNulls 方法: [[${#strings.concatReplaceNulls(user.name, null, 'od', null)}]]

6. 分割与连接

1. 按字符分割 arraySplit

arraySplit 方法:[[${array}]]

2. 按字符连接 arrayJoin 连接数组的每个元素

arrayJoin 方法: [[${#strings.arrayJoin(new String[]{'L', 'u', 'n', 'a'}, '-')}]]

3. listSplit setSplit listJoin setJoin 和上面一致

7. 大小写转换

1. 全部转为大写 toUpperCase ==>SpringBoot

toUpperCase : [[${#strings.toUpperCase('SpringBoot')}]]

2. 全部转为小写 toLowerCase ==>SpringBoot

toLowerCase : [[${#strings.toLowerCase('SpringBoot')}]]

3. 首字母转为大写 capitalize ==>springboot

capitalize : [[${#strings.capitalize('springboot')}]]

4. 首字母转为小写 unCapitalize ==>springboot

unCapitalize : [[${#strings.unCapitalize('SpringBoot')}]]

5. 单词首字母大写 空格区分 capitalizeWords ==>spring boot

capitalizeWords : [[${#strings.capitalizeWords('spring boot')}]]

5. 单词首字母大写 指定符号 '-' capitalizeWords ==>spring boot

capitalizeWords : [[${#strings.capitalizeWords('spring-boot' , '-')}]]

8. 其他处理

1. 截取前后空格 trim

trim: [[${#strings.trim(' Spring Boot ')}]]

2. 省略 abbreviate 只显示个数加上...=指定个数的字符

abbreviate: [[${#strings.abbreviate('SpringBoot', '9')}]]

3. 随机抽取字符 randomAlphanumeric 随机获取5个字符

randomAlphanumeric: [[${#strings.randomAlphanumeric(5)}]]

9. 日期对象

Thymeleaf主要使用org.Thymeleaf.expression.Dates 类处理日期 使用 #dates 标识该对象 0. 格式化日期

[[${#dates.format(user.birth)}]]

1. 年月日时分秒

[[${#dates.format(user.birth, 'yyyy-MM-dd HH:mm:ss')}]]

2. ISO 格式

[[${#dates.formatISO(user.birth)}]]

3. 获取日期字段

获取当前年份: [[${#dates.year(user.birth)}]]

获取当前月份: [[${#dates.month(user.birth)}]]

获取当前天数: [[${#dates.day(user.birth)}]]

获取当前小时: [[${#dates.hour(user.birth)}]]

获取当前分钟: [[${#dates.minute(user.birth)}]]

获取当前毫秒: [[${#dates.second(user.birth)}]]

获取当前月份名称: [[${#dates.monthName(user.birth)}]]

获取当前星期几: [[${#dates.dayOfWeek(user.birth)}]]
